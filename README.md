# Pricing_analytics

this is a pricing analytics project done on cheese_sales data in r.I applied shapiro test to check the normality of the data for natural ad sales category and health sales category and T-test on both the samples to conclude whether population mean is same or not

the data set contains the following

sales_cheese- how many items sold

Price_cheese- price of the item

ad_type- which type of ad it is given(either health or natural product)

price_icecream- icream_price

Price_milk- price of the milk
